\# FW Knight Mask Fix (SPT 4.0.13)



\*\*Fixes the missing "Death Knight" mask texture on Knight PMC and all player models.\*\*



\## Installation

1\. Extract the `FWKnightMaskFix` folder directly into your SPT `user/mods/` folder.

2\. Restart the SPT server.
