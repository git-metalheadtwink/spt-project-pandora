Default map data used is from:
  - TarkovTracker/tarkovData: https://github.com/TarkovTracker/tarkovdata/
  - TarkovDev: https://github.com/the-hideout/tarkov-dev
  - Shebuka: https://github.com/the-hideout/tarkov-dev-svg-maps

Interactive web maps using this data are here: https://tarkov.dev/maps/

All marker data was datamined by this program, but additional info about those markers
gotten from the above sources.
